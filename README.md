# Canvas Diagram

A zero-dependency interactive architecture diagram editor that runs entirely in the browser. No build step, no npm, no frameworks — just open `index.html`.

## Features

- Drag-and-drop nodes with pan & zoom (mouse wheel)
- Bezier curve edges with arrows and labels
- Add **components**, **notes**, and **relationships**
- Color picker per node, snap-to-grid, double-click to edit
- Right-click context menu, selection info box
- Persists to `localStorage` automatically

## Usage

```
open index.html
```

That's it.
